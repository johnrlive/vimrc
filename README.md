# My Vim setup

## Clone Repo into Home directory
```
cd ~
git clone https://github.com/johnrlive/vimrc.git
cd vimrc
```

## To install on Mac run...
```
sh vim_install_mac.sh
```

## To Install on Ubuntu run...
```
sh vim_install_ubuntu.sh
```

## Install Plugs
```
$ vim
:PlugInstall
```
